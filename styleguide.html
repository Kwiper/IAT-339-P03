Portfolio Style Guide

In-Text Link

CSS Selector:

.text-link

Sample Code:

<a href="#" class="text-link">Text</a>

Rendered Element:

This is a link in text

Navigation link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Default nav</a>

Rendered element:

DEFAULT NAV

Image as a link

CSS selector:

.img-link

Sample code:

<a href="#"><img src="img/pokemon_legends_arceus" alt="Test image" class="img-link"></a>

Rendered element:

Headings
